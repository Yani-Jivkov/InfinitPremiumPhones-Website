# InfinitPremiumPhones – Responsive Premium Phone Showcase Website

## 📌 Overview

InfinitPremiumPhones is a responsive front-end website designed to present premium refurbished smartphones in a modern and visually attractive way. The system focuses on high-quality second-hand devices and provides an engaging user experience across desktop and mobile devices.

The website demonstrates modern UI design, responsive layout, animations, and interactive components.

## 🎯 Purpose


The main goal of this project is to create a professional and visually appealing product showcase platform that can be extended into a full e-commerce system in the future.

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
- Responsive Design
- Animations and Transitions
- Modular Multi-page Architecture
